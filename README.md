
# Uber Trip Analysis Dashboard (Power BI) | Data Analyst Portfolio Project

## Overview
This repository contains a **Power BI dashboard project** focused on analysing Uber trip data to derive insights on **bookings, revenue, trip efficiency, time-based demand, and location performance**.  
The project is designed as a **fresher-friendly portfolio project** while maintaining **professional and recruiter-ready standards**.

**Project Type:** Data Analyst Portfolio Project  
**Experience Level:** Fresher / Entry-Level  
**Tools Used:** Power BI, DAX, Data Modeling, Data Visualization

---

## Business Problem
Ride-hailing businesses generate large volumes of trip data but often lack clear visibility into:
- Booking and revenue trends
- Peak demand periods
- Trip efficiency (distance and duration)
- Location-wise and vehicle-wise demand patterns

This project transforms raw trip data into **interactive dashboards** to support **data-driven decision-making**.

---

## Key KPIs
- Total Bookings  
- Total Booking Value (Revenue)  
- Average Booking Value  
- Total Trip Distance  
- Average Trip Distance  
- Average Trip Time  

---

## Dashboard Breakdown

### 1. Overview Analysis Dashboard
**Purpose:** Provide a high-level summary of business performance.

**Key Features:**
- Dynamic KPI selection using **Disconnected Tables**
- Booking and revenue analysis by:
  - Payment Type (Card, Cash, Wallet)
  - Trip Type (Day / Night)
- Vehicle Type performance comparison using Matrix visuals
- Daily booking trend analysis
- Interactive slicers (Date, City)
- Conditional formatting for KPI insights

---

### 2. Time Analysis Dashboard
**Purpose:** Analyse ride demand patterns across time.

**Visuals Included:**
- 10-minute pickup interval analysis (peak vs off-peak hours)
- Day-wise booking trends (weekday vs weekend)
- Hour vs Day heatmap for demand concentration

**Business Value:**
- Supports driver allocation planning
- Helps identify surge pricing windows

---

### 3. Detailed Data View (Drill-Through Page)
**Purpose:** Enable detailed data exploration.

**Features:**
- Trip-level detailed table
- Drill-through from summary dashboards
- Bookmark-enabled toggle between filtered and full dataset
- Improves transparency and analytical depth
